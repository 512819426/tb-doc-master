# Military and Classified Project Guide

## Trigger

Use this guide for projects mentioning:

- 军工
- 涉密
- 保密
- 密级
- 军队
- 武器装备
- 国防
- 国产化
- 信创
- 安全审查
- 涉密信息系统
- 保密资格
- 受控交付

## Core checks

Extract and verify:

- Required confidentiality qualification type and level
- Whether the bidder, subcontractor, manufacturer, or personnel must hold specific qualifications
- Certificate validity and scope
- Classified personnel requirements
- Classified carrier creation, transfer, storage, use, destruction, and archiving requirements
- On-site implementation requirements
- Network, terminal, storage, removable media, and printing control requirements
- Domestic substitution or supply chain control requirements
- Restrictions on outsourcing, subcontracting, remote access, and cloud services
- Security inspection, confidentiality review, and acceptance requirements

## Drafting rules

- Do not invent confidentiality qualifications.
- Do not claim classified project experience without proof.
- Do not disclose sensitive details from prior projects.
- Do not over-describe security controls in a way that creates unnecessary exposure.
- Use "按采购文件要求及采购人管理规定执行" only when the commitment is general and safe.
- For specific commitments, mark for user confirmation.
